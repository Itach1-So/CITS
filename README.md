## 📖 What is CITS?

CITS is a **simple**, **elegant**, and **educational** programming language designed to be easy to learn and fun to use. It combines the simplicity of BASIC with modern programming concepts like functions and structured programming.

> **"CITS - Where Code Meets Simplicity"**

### ✨ Why CITS?

- 🎯 **Beginner Friendly** - English-like syntax that's easy to read and write
- 🔧 **Feature Rich** - Variables, loops, conditions, functions, and more
- 🚀 **Fast Execution** - Written in Python for quick prototyping
- 📚 **Educational** - Perfect for learning programming concepts
- 🌍 **Cross Platform** - Works on Windows, macOS, and Linux
- 💡 **No Dependencies** - Just pure Python, no external libraries needed
